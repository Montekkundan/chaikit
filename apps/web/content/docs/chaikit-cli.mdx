## Template

<Steps>

<Step>Select your project</Step>

Search all listed templates with the following command:

```bash
npx chaikti sip -l
```

<Step>Install the template:</Step>

```bash
npx chaikit sip <template-name>
```


<Step>You're done.</Step>

</Steps>
